# USB Controller for GateMate FPGA

<img src="/doc/gmm-usb-blue-black.png" width="20%">

## Reference Hardware Platform

- GMM-7550 module with USB 3 Adapter

## Full Speed (12 Mbps)

- FPGA I/O and STUSB03E USB Transceiver

## High Speed (480 Mbps)

- ULPI

## Super Speed (5 Gbps)

- GateMate SerDes
